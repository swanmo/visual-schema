SCHEMA BASICS ABOUT

XML Schemas are use for validation and specification of xml messages. Here are some basic examples that will get you started.

Element

Elements are absolutely essential. You'll need them. [Example 1] shows the most basic xsd you can imagine. It allows for an xml document with only on tag, called name.

<?xml version="1.0" encoding="UTF-8" ?>
<xs:schema xmlns:xs="http://www.w3.org/2001/XMLSchema">
<xs:element name="name" type="xs:string"/>
</xs:schema>
