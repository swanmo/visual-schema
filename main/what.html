What is the (eye)xsd editor?

Simple validation

  • Validation of XSD on a basic level
    • Checks that nesting of tags is valid
    • Checks that occurences of attributes are valid
    • Does not validate types of attributes
  • Provides a little bit of help when fabricating an XSD.

Visualization of XSD's

  • Graphical view of the XML Schema
    • Bird's eye view of XML schema.
    • Illustration of all commonly used XSD elements.
      • element (with maxOccurs/minOccurs printed and dashed if optional)
      • attribute (with icon)
      • sequence indicated by ... (symbol)
      • complexType is boxed, to illustrate the packaging
      • extension is boxed, with base type and extensions
      • restriciton is boxed, with base type
      • key/keyref-elements displayed with icon
      • selector/field-elements displayed with icons
      • XSD Restrictions of datatypes are displayed
      • Default presentation of other xsd elements
    • Edit'n view tool for XSD
